Gameplay Features

We've built the groundwork for a wide variety of gameplay features so you don't have to. Adding them to your game is simple with the Steamworks API.
Please refer to the Feature Documentation for more details.

Basic Features

Games of most genres will benefit from these features, which address basic needs. Steamworks API integration is required, but fairly easy to implement.

Achievements

Players expect in-game achievements. Use them to reward dedicated fans, mark special events, and encourage particular activities.

 Game Statistics

 Leaderboards

 Game servers
